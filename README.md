# Sales Analysis – KairoShop

## Project Overview

This project analyzes retail sales data from a clothing business (KairoShop) to identify trends, customer behavior, and opportunities to increase revenue.

The goal is to transform raw data into actionable business insights that support decision-making.

---

## Objective

* Understand sales performance
* Identify top-selling products
* Analyze customer purchasing patterns
* Detect opportunities to improve revenue

---

## Tools Used

* Microsoft Excel
* SQL
* Power BI

---

## Dataset

The dataset represents retail sales transactions and includes:

* Order ID
* Date
* Product
* Category
* Quantity
* Price
* Total Sales
* Customer ID

Note: Data is simulated for analysis purposes.

---

## Analysis Process

1. Data cleaning (handling missing values, formatting)
2. Data exploration using Excel / SQL
3. Aggregations and calculations
4. Data visualization

---

## Key Insights

* A significant percentage of sales occurs during weekends
* A small group of products generates the majority of revenue
* Certain categories consistently outperform others
* There is an opportunity to increase average ticket value

---

## Business Recommendations

* Focus on top-performing products
* Increase promotions during high-demand days
* Create product bundles
* Optimize inventory based on demand

---

## Conclusion

This project demonstrates how data analysis can support business decisions in a retail environment.
